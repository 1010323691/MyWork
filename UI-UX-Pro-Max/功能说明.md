# UI UX Pro Max - 功能说明

## 概述

UI UX Pro Max 是一个 AI 技能插件，提供跨多个平台构建专业用户界面和用户体验的设计智能。其核心创新是**设计系统生成器**，能够分析项目需求并自动产生完整的、定制化的设计系统。

---

## 核心功能（v2.0）

### 主要能力
- **67 种 UI 风格**: Glassmorphism（玻璃态）、Claymorphism（粘土态）、Minimalism（极简主义）、Brutalism（粗野主义）、Bento Grid（便当网格）、Dark Mode（暗色模式）、AI-Native UI 等
- **161 个配色方案**: 与产品类型一一匹配的特定行业配色方案
- **57 种字体组合**: 精心策划的排版组合，附带 Google Fonts 导入
- **24 个落地页模式**
- **161 条推理规则**: 针对特定行业的设计系统生成规则

### 设计系统生成器（旗舰功能）

推理引擎在以下领域执行多域并行搜索：
1. 产品类型匹配（161 种分类）
2. 风格推荐（67 种风格）
3. 配色方案选择（161 个方案）
4. 落地页模式（24 种模式）
5. 排版组合（57 种组合）

输出包括：模式、风格、颜色、排版、效果、需要避免的反模式和交付前检查清单。
